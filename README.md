Welcome to "Knowledge Graph with Neo4j, Cypher, and GDS"! 

This comprehensive course is your gateway to mastering the powerful world of graph databases, a cutting-edge technology reshaping how we handle complex data relationships. Designed for data enthusiasts, developers, and anyone keen on exploring the frontier of data technology, this course will equip you with the skills to build and query robust knowledge graphs using Neo4j.

### What you'll learn
- Graph Database: Neo4j
- Cypher Query Language: Beginner to Advanced Level
- Neo4j and Graph Database Use Cases
- Graph Data Science Library
- Basic Overview of GraphQL
- Overview of Bloom
- Neo4j-AI Integration- Create a Knowledge graph from Unstructured Data using OpenAI LLM
- Overview of RAG and Graph RAG - Knowledge graph as RAG
